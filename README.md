Leaflet Map Project
A simple interactive map using the Leaflet.js library, displaying geographic locations with customizable markers and zoom functionalities.

Description
This project demonstrates the use of Leaflet.js to create interactive maps. It utilizes OpenStreetMap tiles and allows for panning, zooming, and placing custom markers on a map. The map is set to a specific latitude and longitude and can be customized to display different locations by adding more markers or changing the view.

Features
Interactive map with zoom and pan functionality.
Customizable markers to show specific locations.
Responsive design that works on both desktop and mobile devices.
Technologies Used
Leaflet.js: JavaScript library for creating interactive maps.
OpenStreetMap: Provides free map tiles for the project.
HTML/CSS: Used for structuring and styling the page.
JavaScript: For map functionality and interactivity.
